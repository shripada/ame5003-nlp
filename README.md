**Two things that catch everyone out:**

1. **Save your own copy.** The moment a lab opens, do **File → Save a copy in Drive**. Work in
   that copy. If you type into the shared notebook, your work disappears when the tab closes.
2. **Run the install cell first, every time.** Colab wipes the machine between sessions, so
   nothing you installed last week is still there. This is normal, not a bug.

## Books

All prescribed by the syllabus. The first two are free online.

- [Speech and Language Processing](https://web.stanford.edu/~jurafsky/slp3/) — Jurafsky & Martin. The main text.
- [Natural Language Processing with Python](https://www.nltk.org/book/) — the NLTK book.
- [Practical Natural Language Processing](https://www.practicalnlp.ai/) — Vajjala et al. ([code](https://github.com/practical-nlp/practical-nlp-code))
- [Natural Language Processing with PyTorch](https://github.com/delip/PyTorchNLPBook) — Rao & McMahan.
- [A Primer on Neural Network Models for NLP](https://arxiv.org/abs/1510.00726) — Goldberg.
